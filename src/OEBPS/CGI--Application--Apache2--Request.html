CGI::Application::Plugin::Apache2::Request — 1.02

Description

This package is just a wrapper around Apache::Request or Apache2::Request to add CGI.pm compatible methods. The interface and usage is exactly the same as CGI.pm for the methods provided.

  • delete()
  • delete_all()
  • cookie()
  • Dump()
  • Vars()
  • escapeHTML()
  • upload()
