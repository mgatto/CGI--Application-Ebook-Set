CGI::Application::FastCGI — 0.02

For using CGI::Application under FastCGI.

Synopsis

# In "WebApp.pm"…
package WebApp;

use base qw(CGI::Application::FastCGI);
sub setup {
    …
}

1;

# In "webapp.fcgi"…
use WebApp;

my $webapp = WebApp->new;
$webapp->run;

Notes

Note that cgiapp_init() will be called only once under lifecycle of FastCGI. setup() will also only be called once. (you should not be doing magical things in 'setup'.) So if you want to do something for every request, you should write the logic in cgiapp_prerun().
