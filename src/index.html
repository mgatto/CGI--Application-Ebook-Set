Introduction

This ebook is a collection of documentation for CGI::Application and commonly associated modules, specially formatted for modern ereaders. Chapters are clearly delineated and custom fonts are embedded. The default stylesheet is derived from chromatic's "Modern Perl" website, Metacpan's web stylesheet and the CSS used in perldoc.perl.org.

I've also added content from annocpan.org, and attributed any annoations which were included in this compilation.

How We Produced It

  1. Convert POD to HTML

    pod2html --infile=Application.pm --outfile="CGI--Application.html" --noindex --podpath=.

  2. Convert to Kindle .mobi (KF8) and EPUB .epub (EPUB3 and EPUB2)

    ebook-convert CGI--Application.html CGI--Application.mobi --chapter=//h1 --chapter-mark=pagebreak 
--insert-blank-line --insert-blank-line-size=1 --smarten-punctuation --mobi-file-type=new --level1-toc=//h:h1
--level2-toc=//h:h2 --level3-toc=//h:h3 --margin-top=0 --margin-right=0 --margin-left=0 --use-auto-toc 
--language=En --authors="Mark Stosberg"&"Sam Tregar" --output-profile=kindle_fire 
--publisher="Michael Gatto - Lisantra Technologies"

  3. Rename extension from .mobi to .azw3 so the newer KF8 format works on the Kindle Touch.
