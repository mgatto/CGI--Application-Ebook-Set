Introduction

This ebook is a collection of documentation for CGI::Application and commonly associated modules, specially formatted for modern ereaders. Chapters are clearly delineated and custom fonts are embedded. The default stylesheet is derived from chromatic's "Modern Perl" website, Metacpan's web stylesheet and the CSS used in perldoc.perl.org.

I've also added content from annocpan.org, and attributed any annoations which were included into this compilation.

The modules chosen may not be the state of the art in Perl application development and may possibly show their age. However, I think this is an approximation of a reasonable toolkit for older, but not too old, Web application development in Perl.

For more state of the art in Perl Web development, read up on Mojolicious, Dancer, Squatting, Plack, Template::Toolkit, HTML::FormFu or HTML::FormHandler and DBIx::Class.

How We Produced It

  1. Convert POD to HTML

    pod2html --infile=Application.pm --outfile="CGI--Application.html" --podpath=.

  2. Convert to Kindle .mobi (KF8)

    ebook-convert CGI--Application.html CGI--Application.mobi 
  --chapter=//h1 
  --chapter-mark=pagebreak 
  --insert-blank-line 
  --insert-blank-line-size=1 
  --smarten-punctuation 
  --mobi-file-type=new 
  --output-profile=kindle_fire 
  --level1-toc=//h:h1
  --level2-toc=//h:h2
  --level3-toc=//h:h3 
  --margin-top=0 
  --margin-right=0 
  --margin-left=0 
  --use-auto-toc 
  --language=En 
  --authors="Mark Stosberg"&"Sam Tregar" 
  --publisher="Michael Gatto - Lisantra Technologies"

  3. Rename extension from .mobi to .azw3 so the newer KF8 format works on the Kindle Touch.
